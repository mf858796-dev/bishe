# Tobii Pro Lab 集成使用指南

## 概述

本系统支持两种工作模式：
1. **Pro Lab 模式（推荐）**：利用 Tobii Pro Lab 进行校准，无需手动贴标记
2. **独立模式**：使用 ArUco 标记进行动态坐标映射（需要在屏幕四角贴标记）

---

## 模式一：使用 Tobii Pro Lab（推荐）

### 优点
- ✅ 无需在屏幕上贴任何标记
- ✅ Tobii 官方校准流程，精度高
- ✅ 支持多用户快速切换
- ✅ 自动处理坐标映射

### 准备工作

1. **安装 Tobii Pro Lab**
   - 从 Tobii 官网下载并安装 Tobii Pro Lab
   - 确保你有有效的许可证（学生可申请教育许可）

2. **连接 Tobii Pro Glasses 3**
   - 开启眼镜
   - 通过 USB 或 Wi-Fi 连接到电脑
   - 在 Pro Lab 中确认设备已识别

### 使用步骤

#### 第一步：在 Pro Lab 中完成校准

1. 打开 Tobii Pro Lab
2. 创建新项目或打开现有项目
3. 添加参与者（Participant）
4. 点击 **Calibration** 标签页
5. 选择 **Screen Calibration**（屏幕校准）
6. 按照屏幕提示完成校准流程：
   - 用户注视屏幕上移动的校准点
   - 通常需要注视 5-9 个点
   - 校准完成后会显示精度评估

7. **验证校准**：
   - 在 Pro Lab 的 Live View 中查看 gaze 点是否准确
   - 如果精度不满意，重新校准

#### 第二步：启动你的训练系统

1. 保持 Pro Lab 运行（不要关闭）
2. 在你的系统中连接眼镜：
   ```bash
   cd e:\毕设
   venv\Scripts\activate
   python main_window.py
   ```
3. 点击"连接 Tobii Glasses 3"按钮
4. 系统会自动检测到已连接的眼镜

#### 第三步：开始训练

- 系统会直接接收来自 Tobii 的 gaze 数据
- Pro Lab 已经完成了坐标映射，所以 gaze 坐标已经是屏幕坐标
- 开始进行代码阅读专注力训练

### 注意事项

⚠️ **重要**：
- Pro Lab 和你的训练系统**不能同时录制**，但可以同时接收 gaze 数据流
- 如果需要录制数据，建议在 Pro Lab 中录制，然后导出数据进行分析
- 每次更换用户或大幅改变坐姿后，需要重新校准

## 模式二：独立模式（使用 ArUco 标记）

2. 将它们贴在显示器的四个角
3. 测量标记的位置以便校准

## 常见问题

### Q1: Pro Lab 和我的系统可以同时运行吗？

**A:** 可以。Tobii Pro Glasses 3 支持多个客户端同时接收 gaze 数据流。但是：
- 不能同时开始录制（只有一个应用可以控制录制）
- 建议在 Pro Lab 中完成校准后，只让你的系统接收 gaze 数据

### Q2: 校准后多久需要重新校准？

**A:** 以下情况需要重新校准：
- 更换用户
- 同一用户但大幅改变坐姿或眼睛位置
- 佩戴了不同的眼镜
- 环境光线发生显著变化

一般建议每次实验 session 开始前都重新校准。

### Q3: 如何验证校准精度？

**A:** 
1. 在 Pro Lab 的 Live View 中观察 gaze 点
2. 注视已知位置（如屏幕角落），检查 gaze 点是否重合
3. 本系统会在训练过程中实时显示 gaze 点，可以直观判断精度

### Q4: 没有 Pro Lab 怎么办？

**A:** 你有以下选择：
1. 联系 Tobii 申请教育许可证（免费或优惠）
2. 使用 ArUco 标记方案（需要修改代码）
3. 实现一个简单的多点校准流程（可以自己开发）

## 附录：Gaze 数据格式

从 Tobii Pro Glasses 3 获取的数据包含：

```python
{
    'gaze2d': [u, v],           # 归一化坐标 (0-1)
    'gaze3d': [x, y, z],        # 3D 坐标 (mm)，相对于场景相机
    'pupil_diameter': 3.5,      # 瞳孔直径 (mm)
    'screen_x': 960,            # 屏幕 X 坐标（如果使用 Pro Lab 校准）
    'screen_y': 540,            # 屏幕 Y 坐标（如果使用 Pro Lab 校准）
}
```

在使用 Pro Lab 时，系统会自动处理坐标转换，你只需要关注 `screen_x` 和 `screen_y`。
