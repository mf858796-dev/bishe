# 编程初学者代码阅读专注力训练系统 - 运行指南

## 📋 项目概述

这是一个基于 Tobii Pro Glasses 3 眼动仪的专注力训练系统,可以实时追踪用户的视线移动,分析阅读代码时的专注度。

## 🖥️ 环境要求

### 硬件要求
- **Tobii Pro Glasses 3** 眼动仪
- Windows 电脑 (推荐 Windows 10/11)
- USB-C 数据线 (用于连接眼镜)

### 软件要求
- Python 3.10+
- PyQt5
- OpenCV
- g3pylib (Tobii SDK)

## 🚀 快速开始

### 步骤 1: 准备工作

1. **确保 Tobii Pro Glasses 3 已充电**
   - 长按电源键 2 秒开机
   - 等待指示灯变为绿色

2. **连接眼镜到电脑**
   - 使用 USB-C 线连接眼镜和电脑
   - 等待 Windows 自动安装驱动

3. **验证连接**
   ```bash
   ping 192.168.75.51
   ```
   或者运行诊断脚本:
   ```bash
   venv\Scripts\python.exe test_gaze_simple.py
   ```

### 步骤 2: 启动应用

#### 方法 A: 使用批处理文件 (推荐)
```bash
双击运行: 启动应用.bat
```

#### 方法 B: 命令行启动
```bash
# 打开命令提示符 (Win + R, 输入 cmd)
cd e:\毕设
venv\Scripts\python.exe main.py
```

### 步骤 3: 连接眼动仪

1. **在应用窗口中**:
   - 找到状态栏的"连接方式"下拉框
   - 选择 **"🔍 自动发现 (推荐)"**

2. **点击连接按钮**:
   - 点击底部的 "🔌 连接 Tobii Pro Glasses 3" 按钮
   - 等待连接成功

3. **连接成功标志**:
   - 状态栏显示: "● 已连接 (TG03B-xxxxxxxxxx)"
   - 视频流开始显示
   - 热力图开始更新

### 步骤 4: 开始训练

1. **切换到"实时监控"标签页**
   - 查看实时视频流
   - 观察视线热力图
   - 监控视线轨迹

2. **切换到"训练任务"标签页**
   - 开始专注力训练
   - 完成指定的阅读任务

3. **切换到"数据分析"标签页**
   - 查看注视统计
   - 分析训练进度
   - 查看详细指标

## 🔧 故障排查

### 问题 1: 连接超时

**症状**: 点击连接后一直显示"连接中..."

**解决方法**:
```bash
# 1. 运行诊断脚本
venv\Scripts\python.exe test_gaze_simple.py

# 2. 检查设备管理器
# Win + X -> 设备管理器 -> 网络适配器
# 查找 Tobii 或 Remote NDIS 设备

# 3. 检查网络连接
ping 192.168.75.51
```

### 问题 2: 找不到设备

**可能原因**:
- 眼镜未开机
- USB 驱动未安装
- 防火墙阻止连接

**解决方法**:
1. 确认眼镜指示灯为绿色
2. 重新插拔 USB 线
3. 暂时关闭防火墙测试
4. 重启眼镜和应用

### 问题 3: ArUco 检测错误

**症状**: `AttributeError: module 'cv2.aruco' has no attribute 'detectMarkers'`

**解决方法**: 已修复!代码已兼容不同版本的 OpenCV。

### 问题 4: 视频流不显示

**可能原因**:
- RTSP 流配置问题
- 防火墙阻止

**解决方法**:
1. 检查 Windows 防火墙设置
2. 允许 Python 和 RTSP 端口 (8554)
3. 重启应用

## 💡 使用建议

1. **首次使用**:
   - 先运行 `test_gaze_simple.py` 确认连接正常
   - 确保眼镜已正确校准

2. **训练时**:
   - 保持头部稳定
   - 自然阅读,不要刻意控制眼球
   - 定期休息,避免眼疲劳

3. **数据分析**:
   - 关注有效注视率 (>70% 为佳)
   - 注意回视次数 (越少越好)
   - 观察视线熵值 (越低越专注)
