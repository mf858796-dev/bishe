## 已知问题与注意事项

1. **g3pylib 安装**：由于 `av` 库在 Windows 上的编译问题，需要修改 `g3pylib/pyproject.toml` 中的依赖版本要求（已将 `av ~= 10.0.0` 改为 `av >= 10.0.0`）

2. **编码问题**：Windows 命令行默认使用 GBK 编码，可能导致中文字符显示异常。建议使用 PowerShell 或 Git Bash。

3. **首次运行**：程序会尝试搜索 Tobii 设备，如果没有连接设备，会显示"未找到设备"的提示，这是正常行为。
