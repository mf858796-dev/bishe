## 📋 优化清单

### ✅ 1. 边缘密集型非均匀网格
**问题**：均匀网格在边缘区域采样不足，头动时边缘误差大
**方案**：
- U/V方向：`[0.05, 0.1, 0.2, 0.35, 0.5, 0.65, 0.8, 0.9, 0.95]`
- 总点数：9×9 = 81个点
- 优势：边缘密度是中心的3倍，更好捕捉头动畸变

**实现位置**：`main_window.py::_run_initial_calibration()`

---

### ✅ 2. 岭回归（Ridge Regression）替代普通最小二乘
**问题**：普通最小二乘在有噪声数据下容易过拟合
**方案**：
```python
from sklearn.linear_model import Ridge

# 二次多项式特征矩阵 X (n x 6)
# 使用岭回归求解，alpha为正则化强度
ridge_u = Ridge(alpha=1.0)
ridge_u.fit(X, Y_u)
coeffs_u = ridge_u.coef_

ridge_v = Ridge(alpha=1.0)
ridge_v.fit(X, Y_v)
coeffs_v = ridge_v.coef_
```

**优势**：
- L2正则化防止过拟合
- 对噪声更鲁棒
- 系数更稳定

**实现位置**：`main_window.py::calculate_polynomial_calibration()`

---

### ✅ 3. RANSAC异常值剔除
**问题**：眨眼、头动产生的异常样本影响拟合精度
**方案**：
```python
from sklearn.linear_model import RANSACRegressor

# RANSAC自动剔除异常值
ransac_u = RANSACRegressor(Ridge(alpha=1.0), 
                           min_samples=0.8,  # 至少80%内点
                           residual_threshold=0.05,  # 残差阈值
                           max_trials=100)
ransac_u.fit(X, Y_u)
coeffs_u = ransac_u.estimator_.coef_
inlier_mask = ransac_u.inlier_mask_  # 内点掩码
```

**预期效果**：误差降低15%

**实现位置**：`main_window.py::calculate_polynomial_calibration()`

### ✅ 5. 实时眼镜滑动检测
**原理**：监测校准误差的突然变化
**方案**：
```python
def detect_glasses_slip(self, current_error, error_history):
    """检测眼镜是否滑动"""
    if len(error_history) < 5:
        return False
    
    # 如果当前误差突然增大超过历史平均的3倍
    avg_error = np.mean(error_history[-10:])
    if current_error > avg_error * 3:
        return True
    return False
```

**触发条件**：
- 连续5次注视点误差 > 阈值
- 误差标准差突然增大

**实现位置**：新增 `main_window.py::monitor_calibration_quality()`

### ✅ 6. 隐式动态校准
**原理**：利用用户自然注视屏幕中心的时刻自动微调
**方案**：
```python
def implicit_calibration(self, gaze_u, gaze_v, screen_center=(0.5, 0.5)):
    """隐式动态校准"""
    # 检测是否注视中心区域（±0.1范围）
    if abs(gaze_u - 0.5) < 0.1 and abs(gaze_v - 0.5) < 0.1:
        # 收集中心区域的注视数据
        self.center_gaze_samples.append((gaze_u, gaze_v))
        
        # 每收集10个样本，微调校准参数
        if len(self.center_gaze_samples) >= 10:
            self.fine_tune_calibration()
            self.center_gaze_samples.clear()
```

**实现位置**：新增 `main_window.py::implicit_calibration_update()`
