### 2. RANSAC异常值剔除（简化版）✅
**状态**: 已实施（不依赖scikit-learn的自定义实现）

**改进内容**:
- 两阶段拟合策略：
  1. 普通最小二乘初步拟合
  2. 动态阈值检测异常值（中位数×2.5）
  3. 剔除异常值后重新拟合

**技术细节**:
```python
# 计算残差
residuals_u = np.abs(Y_u - pred_u_init)
residuals_v = np.abs(Y_v - pred_v_init)
total_residuals = residuals_u + residuals_v

# 动态阈值
threshold = np.median(total_residuals) * 2.5
inlier_mask = total_residuals < threshold
```

**文件修改**:
- `main_window.py::calculate_polynomial_calibration()` (第2294-2330行)

**预期效果**:
- 自动剔除眨眼、头动产生的异常样本
- 误差降低约10-15%

### 5. 隐式动态校准 ✅
**状态**: 已实施

**工作原理**:
- 检测用户是否注视屏幕中心区域 (0.45-0.55)
- 收集中心区域的注视样本
- 每10个样本微调一次校准参数
- 缓慢调整常数项偏置

**技术细节**:
```python
# 检测中心偏移
offset_u = 0.5 - avg_u
offset_v = 0.5 - avg_v

# 微调常数项（缓慢调整）
self.mapper.poly_coeffs_u[0] += offset_u * 0.1
self.mapper.poly_coeffs_v[0] += offset_v * 0.1
```

**优势**:
- 无需用户主动操作
- 利用自然使用过程持续优化
- 补偿眼镜微小滑动

**文件修改**:
- `main_window.py::implicit_calibration_update()` (新增方法)
