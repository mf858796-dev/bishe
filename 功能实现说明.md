## 🔧 核心算法与数据处理

### 1. **单应性矩阵校准**
```python
# 将归一化坐标转换为像素坐标
dst_points = np.column_stack([
    screen_u * self.mapper.screen_width,
    screen_v * self.mapper.screen_height
]).astype(np.float32)

H, mask = cv2.findHomography(src_points, dst_points, cv2.RANSAC, 5.0)
```

### 2. **注视点映射**
```python
# 使用单应性矩阵进行透视变换
predicted = cv2.perspectiveTransform(
    src_points.reshape(-1, 1, 2), H
).reshape(-1, 2)
```

### 3. **专注度评估**
```python
# 多维度综合评分
score = fixation_rate - regression_penalty - entropy_penalty + data_quality
score = max(0, min(100, score))
```

## 🔍 故障排查

### 常见问题：

1. **注视点不显示**
   - 检查眼动仪是否连接
   - 确认校准是否完成
   - 查看"显示注视点"复选框是否勾选

2. **数据接收率为0**
   - 检查网络连接
   - 确认RTSP地址正确
   - 查看防火墙设置

3. **报告无法生成**
   - 检查reports目录权限
   - 确认record数据完整
   - 查看控制台错误信息

4. **主题切换无效**
   - 点击"保存设置"按钮
   - 重启应用程序
   - 检查settings.json是否正确保存
