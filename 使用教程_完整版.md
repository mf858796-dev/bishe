## 🎯 使用流程（5 步完成）

### 步骤 1：启动 Tobii Pro Glasses 3

1. **开启眼镜**
   - 长按电源键 2 秒
   - 等待指示灯变为绿色（表示已开机）

2. **连接到电脑**
   
   **方式 A：Wi-Fi 连接（推荐）**
   - 眼镜会自动创建 Wi-Fi 热点
   - SSID: `TG03B-xxxxxxxxxx`（你的序列号）
   - 密码: `TobiiGlasses`
   - 在你的电脑上连接这个 Wi-Fi

   **方式 B：USB 连接**
   - 使用 USB-C 线连接眼镜和电脑
   - 等待驱动程序自动安装

3. **验证连接**
   - 打开浏览器
   - 访问 `http://192.168.75.51`（Wi-Fi 模式）
   - 如果能看到 Tobii 网页界面，说明连接成功

### 步骤 2：在 Tobii Pro Lab 中完成校准

> ⚠️ **这是最关键的一步！** 校准质量直接影响训练效果。

1. **打开 Tobii Pro Lab**
   - 双击桌面上的 Tobii Pro Lab 图标
   - 等待软件加载

2. **创建新项目**
   ```
   File → New Project
   项目名称: CodeReading_Training
   保存位置: e:\毕设\prolab_data
   ```

3. **添加参与者（你自己）**
   ```
   Participants → Add Participant
   姓名: 孟祥运
   ID: 202211040567
   ```

4. **设置显示器**
   ```
   Settings → Display Settings
   - 选择你的显示器型号
   - 或手动输入尺寸（例如：24 英寸，1920x1080）
   - 测量并输入眼睛到屏幕的距离（通常 60-70 cm）
   ```

5. **开始校准**
   ```
   Calibration → Screen Calibration
   ```
   
   **校准过程：**
   - 坐在正常阅读代码的位置
   - 保持头部自然（不要刻意固定）
   - 用眼睛跟随屏幕上的校准点
   - 校准点会依次出现在不同位置
   - 每个点注视 1-2 秒，直到它消失
   
   **校准点数量：**
   - 5 点校准（快速，精度一般）
   - 9 点校准（推荐，平衡速度和精度）
   - 13 点校准（最精确，但耗时较长）

6. **验证校准精度**
   - 校准完成后，Pro Lab 会显示精度报告
   - **优秀**: < 0.5° 视觉角度
   - **良好**: 0.5° - 1.0°
   - **需要重校**: > 1.0°
   
   如果精度不理想，点击 "Recalibrate" 重新校准。

7. **测试 Live View**
   ```
   Live View → Start
   ```
   - 看着屏幕上的不同位置
   - 观察红色 gaze 点是否跟随你的视线
   - 如果偏差较大，需要重新校准

1. **点击连接按钮**

2. **等待连接**
   - 状态栏会显示: "● 连接中..."（黄色）

   - 继续按照提示完成训练

5. **查看实时指标**
   - 注视代码时，指标卡片会实时更新
   - 有效注视率越高越好（目标 > 80%）
   - 回视次数越少越好
   - 视线熵越低表示注视越稳定

   - **视线熵**: 视线跳跃的混乱程度（越低越专注）

## 💡 常见问题解答

### Q1: 连接时显示"未找到设备"怎么办？

**A:** 检查以下几点：
1. 眼镜是否已开机（指示灯是否为绿色）
2. 电脑是否已连接到眼镜的 Wi-Fi
3. 防火墙是否阻止了连接
4. 尝试重启眼镜和程序
5. 检查虚拟环境是否正确激活

### Q2: 校准总是不准确怎么办？

**A:** 注意以下技巧：
1. **保持合适的距离**: 眼睛距离屏幕 60-70 cm
2. **光线充足**: 避免强光直射眼睛或屏幕
3. **摘掉反光眼镜**: 如果戴眼镜，尽量去除反光
4. **自然坐姿**: 不要刻意保持僵硬姿势
5. **多次校准**: 多试几次，取最好的一次

### Q3: gaze 点抖动很厉害怎么办？

**A:** 这是正常现象，系统已经添加了平滑滤波。如果仍然抖动：
1. 检查校准精度是否足够
2. 确保环境光线稳定
3. 避免频繁眨眼
4. 保持头部相对稳定

### Q4: 训练数据保存在哪里？

**A:** 
- 热力图和轨迹图保存在 `e:\毕设\` 目录
- 文件名格式: `level_X_heatmap.png` 和 `level_X_trajectory.png`
- 可以在 Tobii Pro Lab 中导出更详细的原始数据

### Q5: 如何判断训练效果好不好？

**A:** 参考以下标准：
- **有效注视率**: > 70% 为良好，> 85% 为优秀
- **回视次数**: < 3 次为良好，< 1 次为优秀
- **视线熵**: < 2.0 为良好，< 1.0 为优秀
- **任务完成时间**: 越短越好

### Q6: 可以同时运行 Tobii Pro Lab 和训练系统吗？

**A:** 可以。Tobii Pro Lab 用于校准，校准完成后可以保持运行或关闭，训练系统会直接从眼镜获取数据。建议校准后关闭 Pro Lab 以节省系统资源。

### Q7: 训练过程中可以暂停吗？

**A:** 当前版本暂不支持暂停功能。如果需要暂停，可以点击"断开连接"按钮，下次重新连接后从当前关卡继续。
